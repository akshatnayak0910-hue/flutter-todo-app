# Flutter To-Do App

A simple and modern To-Do App built using Flutter and Dart.

This project demonstrates the basic concepts of Flutter development such as:

* StatefulWidget
* TextEditingController
* Add, Edit, and Delete task functionality
* AlertDialog usage
* ListView.builder
* FloatingActionButton
* UI design using Card, ListTile, Stack, and Scaffold

## Features

✔ Add new tasks
✔ Edit existing tasks
✔ Delete tasks
✔ Task list display using ListView.builder
✔ Floating Action Button for quick task addition
✔ AlertDialog for task input
✔ Clean and modern UI
✔ Responsive design
✔ Beginner-friendly Flutter project

## Technologies Used

* Flutter
* Dart
* Material Design

## Output Screen

The app contains:

* AppBar with title
* Task list display
* Add task dialog
* Edit task dialog
* Delete task option
* Floating action button
* Empty task message ("No tasks yet!")
* Custom footer text
